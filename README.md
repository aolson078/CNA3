# The Campaign for North Africa — Rules Reference

Rules for **The Campaign for North Africa** (SPI, 1979), converted from the original rulebook into organized Markdown files for easy reference and searchability.

CNA is a legendary wargame simulating operations in Libya and Egypt from 1940 through early 1943. It is widely considered the most complex board game ever published, with a particular emphasis on logistics. The game is designed for 8-10 players (minimum 5 per side) and uses SPI's "Case System" for rule organization (e.g., `[8.12]` = Section 8, Primary Case 1, Secondary Case 2).
